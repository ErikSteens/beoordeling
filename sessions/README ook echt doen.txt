Je kunt het session path dus aanpassen. Normaal wordt dat in een standaard folder/map ergens op de server opgeslagen.
Het is prima als je het per project / per webpagina gaat bijhouden.
Maar dan mag het natuurlijk niet in de rootfolder staan. Want nu kunt je deze map via de browser gewoon benaderen.
En dat wil je zeer zeker niet.
Dus het is verstandig deze folder te maken een stapje boven je rootmap. Maar voor het voorbeeld nu laten we het even zo.
Dus denk hier goed over na als je er mee aan de slag gaat.
